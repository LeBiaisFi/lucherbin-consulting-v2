# Luc Herbin Consulting V2

Landing page statique prête à être mise en ligne.

## Lancer en local

```bash
python3 -m http.server 4173
```

Puis ouvrir http://localhost:4173.

## Déploiement rapide

Le projet est 100% statique (`index.html`), donc il peut être publié sur n'importe quel hébergeur statique :

- Netlify (drag-and-drop du dossier)
- Vercel (import du repo)
- GitHub Pages
- Cloudflare Pages

Aucune étape de build n'est nécessaire.
